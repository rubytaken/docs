## Available models

All models are available on the free tier with daily usage limits. Switch between models using the model selector in the chat header.

## How to choose a model

| Use case | Recommended model |
|----------|-------------------|
| Quick price checks and simple questions | Gemini 3.1 Flash Lite |
| Complex analysis with multiple tools | GPT-4.1 (Azure) |
| General daily usage | GPT-4o Mini (Azure) |
| Image/chart analysis | Pixtral Large |
| Code-related questions | Devstral Small |
| Casual conversation | Any free tier model |

<Tip>The default model is **Gemini 3.1 Flash Lite** — it offers the best speed-to-quality ratio for most crypto queries.</Tip>
