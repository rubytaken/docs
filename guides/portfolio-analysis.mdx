## Before you start

To get the most from portfolio analysis, connect your wallet first. See [Wallet integration](/features/wallet-integration) for setup instructions.

You can also analyze any public wallet by pasting its address directly in the chat.

## Analyze your portfolio

<Steps>
  <Step title="Connect your wallet or paste an address">
    Connect via the wallet icon, or type:
    ```
    Analyze this wallet: YOUR_WALLET_ADDRESS
    ```
  </Step>

  <Step title="Get a portfolio overview">
    Ask for a broad analysis first:
    ```
    Give me a full breakdown of my portfolio
    ```

    The agent returns:
    - Token balances with USD values
    - Portfolio allocation percentages
    - 24h change for each holding
    - Total portfolio value
  </Step>

  <Step title="Check risk and diversification">
    ```
    Am I too heavy on one coin?
    ```
    ```
    What's my portfolio risk score?
    ```

    The **Portfolio Advisor** agent assesses:
    - Concentration risk (percentage in top holdings)
    - Sector diversification (DeFi, L1s, meme coins, stablecoins)
    - Correlation between holdings
    - Overall risk rating
  </Step>

  <Step title="Get rebalancing suggestions">
    ```
    Should I rebalance my portfolio?
    ```

    The agent provides specific suggestions:
    - Which tokens to reduce or increase
    - Target allocation percentages
    - Reasoning based on current market conditions
  </Step>
</Steps>

## Multi-chain analysis

For wallets active across multiple chains:

```
Scan my wallets across all chains
```

The **Comprehensive Analyst** agent uses multi-chain wallet analysis tools to aggregate your holdings across EVM chains and Solana into a unified view.

## Solana-specific analysis

The **Solana Wallet Advisor** agent provides deep Solana analytics:

```
Check my Solana NFTs and DeFi positions
```

This returns:
- SPL token balances
- Active DeFi positions (Jupiter, Orca, Raydium, Kamino)
- NFT collection breakdown with rarity scores
- Staking account status
- Recent transaction summary

## Track performance over time

Use chat history to compare your portfolio across different dates. Ask the same analysis query periodically to track:

- Portfolio value changes
- Allocation drift
- Whether rebalancing suggestions played out

<Tip>Combine portfolio analysis with market trend queries for context. For example: "My portfolio is heavy on SOL — is the Solana ecosystem growing or shrinking?"</Tip>
