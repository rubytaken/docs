## The chat interface

When you open heyRISA, you see the main chat interface with:

- **Model selector** — choose your AI model from the dropdown in the header
- **Agent selector** — pick a specialized agent or use the default chat
- **Suggestion categories** — pre-built prompts organized by topic
- **Chat input** — type your question in natural language

## Select a model

Click the model dropdown in the chat header to switch between available AI models. Each model has different strengths:

- **Gemini 3.1 Flash Lite** (default) — fastest responses for quick queries
- **GPT-4.1 (Azure)** — best for complex, multi-step analysis
- **Pixtral Large** — use when you need image understanding

See [Supported models](/features/supported-models) for the full list.

## Use suggestion categories

The home screen shows six suggestion categories, each with four ready-to-use prompts:

| Category | What it covers |
|----------|---------------|
| Real-time market analysis | Fear & Greed, trending tokens, top gainers/losers |
| Portfolio analysis | Wallet scanning, rebalancing, multi-chain analysis |
| Trading signals | Buy/sell signals, support/resistance, price prediction |
| News & events | Headlines, token unlocks, sentiment analysis |
| NFT & DeFi | Yield farming, NFT evaluation, protocol comparison |
| Strategy & optimization | Allocation advice, DCA strategies, risk management |

Click any suggestion to start a chat with the preferred agent for that query type.

## Write effective prompts

Agents respond best when you're specific about what you need.

<Tabs>
  <Tab title="Good prompts">
    ```
    Give me a technical analysis of BTC with RSI and MACD on the 4h timeframe
    ```
    ```
    Compare yield farming APYs on Jupiter vs Orca for USDC
    ```
    ```
    Analyze wallet 7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU — what should I sell?
    ```
  </Tab>
  <Tab title="Vague prompts">
    ```
    Tell me about crypto
    ```
    ```
    What should I buy?
    ```
    ```
    Is it bullish?
    ```
  </Tab>
</Tabs>

<Tip>Include specific tokens, timeframes, and what kind of analysis you want. The more context you give, the better the response.</Tip>

## File uploads

Some models support file uploads. You can upload:

- Screenshots of charts for visual analysis
- CSV files with portfolio data
- Images of trading patterns for identification

Check the model card in the selector to see if file upload is supported.

## Chat history

All conversations are saved automatically. Access your chat history from the sidebar to:

- Continue previous conversations
- Reference past analysis
- Track how signals played out over time

## Related questions

After each response, agents suggest related follow-up questions. Click any suggestion to continue the analysis without typing.
